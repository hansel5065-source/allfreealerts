# AllFreeAlerts.com — Launch Plan

## Phase 1: Foundation (This Week)
**Goal: Get the site live and working**

### Day 1-2: Technical Setup
- [ ] Set up Cloudflare account + connect allfreealerts.com domain
- [ ] Deploy site to Cloudflare Pages
- [ ] Set up Cloudflare Analytics (free, no cookie banner needed)
- [ ] Add Google Analytics 4 (free, more detailed data)
- [ ] Set up scraper to run daily (Windows Task Scheduler or GitHub Actions)
- [ ] Test everything works: scraper → data.json → live site updates

### Day 3-4: Email System
- [ ] Set up free email service (Mailchimp free tier = 500 contacts, or Brevo free = 300 emails/day)
- [ ] Create welcome email: "Here's what you'll get every day"
- [ ] Create daily digest template: "12 new freebies found today"
- [ ] Connect email signup form on site to the email service
- [ ] Set up automated daily email (scraper finishes → triggers email)

### Day 5: Content Prep
- [ ] Write "About" page: who we are, what we do, why trust us
- [ ] Write 3-5 TikTok/Reel scripts for launch week
- [ ] Create simple logo (Canva free tier)
- [ ] Set up Instagram account (@allfreealerts)
- [ ] Set up TikTok account (@allfreealerts)

---

## Phase 2: Launch & First Users (Weeks 2-3)
**Goal: Get first 100 visitors and 25 email subscribers**

### Social Media Launch
- [ ] Post first TikTok/Reel: "I built a site that finds every freebie on the internet"
- [ ] Post daily: "3 things you can get for FREE today" (pick best items from scraper)
- [ ] Post settlement alerts: "You might be owed $500 — here's how to claim it" (these go viral)
- [ ] Engage in Reddit r/freebies, r/sweepstakes (share genuinely, don't spam)
- [ ] Share in Facebook freebie/sweepstakes groups (there are dozens with 100K+ members)

### Content Strategy (what to post)
- **TikTok/Reels (3-5x per week):**
  - "3 FREE things you can get right now"
  - "This company owes you money and you don't even know" (settlement alerts)
  - "I entered 10 sweepstakes today — here's how"
  - "Free stuff haul — everything I got this week for $0"
  - "Websites that give you free stuff" (mention AllFreeAlerts)

- **Email newsletter (daily):**
  - Subject line: "🎁 [number] new freebies today + [highlight item]"
  - Top 3 ending soon
  - New sweepstakes
  - New freebies
  - New settlements worth claiming
  - One-liner CTA: "See all [number] on AllFreeAlerts.com"

### Track Everything
- [ ] Check Cloudflare Analytics daily: visitors, page views, countries
- [ ] Check email open rates and click rates
- [ ] Track which social posts get the most engagement
- [ ] Track which categories people click most on the site

## Phase 3: Grow (Weeks 4-8)
**Goal: 1,000 daily visitors, 200 email subscribers**

### SEO (free long-term traffic)
- [ ] Add individual pages for each category (/sweepstakes, /freebies, /settlements)
- [ ] Write blog posts targeting search keywords:
  - "Best sweepstakes to enter today [month] 2026"
  - "Free samples by mail [month] 2026"
  - "Open class action settlements 2026"
  - "How to get free stuff online"
- [ ] Submit sitemap to Google Search Console
- [ ] These pages will rank in Google and bring passive traffic

### Affiliate Setup
- [ ] Apply to ShareASale (sweepstakes/deals affiliate programs)
- [ ] Apply to CJ Affiliate
- [ ] Apply to Impact (has sweepstakes brands)
- [ ] Add affiliate links where applicable (some sweepstakes platforms pay per signup)
- [ ] Track clicks and conversions

### Partnerships
- [ ] Reach out to 5 sweepstakes bloggers for cross-promotion
- [ ] Reach out to freebie influencers on TikTok/Instagram for shoutouts
- [ ] Guest post on freebie blogs linking back to AllFreeAlerts

## Phase 4: Monetize (Months 2-4)
**Goal: First revenue**

### Revenue Stream 1: Display Ads
- [ ] Apply for Google AdSense (need ~1000 daily visitors)
- [ ] Or use Mediavine/Raptive (need 50K+ monthly sessions — longer term goal)
- [ ] Expected: $5-15/day at 1K daily visitors with AdSense

### Revenue Stream 2: Sponsored Listings
- [ ] Create a "Featured" tier on the site (pinned card at top, highlighted border)
- [ ] Reach out to brands running sweepstakes: "Want to be featured to our [X] daily visitors?"
- [ ] Price: $50-200/week per featured listing (depends on traffic)
- [ ] Start with 1-2 sponsors

### Revenue Stream 3: Email Sponsors
- [ ] Once you have 500+ subscribers, sell a sponsored slot in the daily email
- [ ] "Today's Featured Sweepstakes" — presented by [Brand]
- [ ] Price: $25-100 per email send (depends on list size)

### Revenue Stream 4: Affiliate Commissions
- [ ] Track which affiliate links convert
- [ ] Double down on the ones that pay
- [ ] Expected: $100-500/month once traffic is steady

---

## Phase 5: Scale (Months 4-12)
**Goal: 10K+ daily visitors, $1K+/month revenue**

### More Sources
- [ ] Add more scraper sources as they're discovered
- [ ] Add user-submitted sweepstakes/freebies
- [ ] Add browser extension that alerts users about freebies on sites they visit
- [ ] Add mobile app (PWA — progressive web app, free to build)

### Premium Tier ($3-5/month)
- [ ] Early access to new items (6 hours before free users)
- [ ] No ads
- [ ] "Winning tips" content
- [ ] Exclusive high-value sweepstakes
- [ ] Entry reminders for daily-entry sweepstakes

### Reach Out to Brands Directly
- [ ] Build a media kit: traffic stats, demographics, engagement rates
- [ ] Contact brand marketing departments:
  - "We have [X] daily visitors actively looking for sweepstakes and freebies"
  - "Feature your sweepstakes to our engaged audience"
  - "Our users have a 40% email open rate" (freebies emails tend to have high opens)
- [ ] Target: P&G, General Mills, Coca-Cola, Kellogg's sweepstakes departments
- [ ] Also target: smaller DTC brands that run giveaways for visibility
